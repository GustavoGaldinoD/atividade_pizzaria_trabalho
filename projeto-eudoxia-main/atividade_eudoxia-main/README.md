# Sobre Meu Projeto

Este projeto tem como objetivo oferecer soluções inovadoras para diminuir casos de despedicio de comida.

## Funcionalidades

- Cadastro de usuários
- Cadastro de empresas
- Dashboard interativo
- Desing simples e moderno
